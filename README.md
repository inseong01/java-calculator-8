# java-calculator-precourse

## 문제 정리
입력한 문자열에서 구분자 통해 추출한 숫자를 더하는 계산기다.

잘못된 문자열을 입력하면 메시지가 전달되는 동시에 오류를 발생시켜 프로그램을 종료시킨다.

### 구분자

**기본 구분자**  
쉼표 또는 콜론

예시
```markdown
""        =>  결과값: 0
"1,2"     =>  결과값: 3
"1,2,3"   =>  결과값: 6
"1,2:3"   =>  결과값: 6
```

**커스텀 구분자**  
"//" 와 "/n" 사이 위치하는 문자

예시
```markdown
"//;\n1;2;3" -> 결과값: 6 / 커스텀 구분자: ";"
"//@\n1@2@3" -> 결과값: 6 / 커스텀 구분자: "@"
```

### 메시지
메시지는 "[ERROR]"로 시작한다.

**조건**

사용자가 잘못된 값을 입력하는 경우 메시지를 전달한다.

**추가기능**

메시지가 출력되면 Error도 함께 발생하며 프로그램을 종료시킨다.

**입력**
구분자와 양수로 구성된 문자열이다.

## 개발

### 애플리케이션 흐름

**올비른 문장인 경우**

```markdown
1. 사용자가 문장을 입력한다.
2. 구분기에서 문장을 검증한다.
3. 구분기에서 구분자를 추출한다.
4. 구분기에서 숫자를 추출한다.
5. 계산기에서 숫자를 더한다.
6. 숫자를 출력한다.
```

**올바르지 않은 문장인 경우**

```markdown
1. 사용자가 문장을 입력한다.
2. 구분기에서 문장을 검증한다. (오류 발생)
3. 프로그램을 종료시킨다.
```

### 기능 목록

`입력`  
1. 계산 문자 입력

`출력`
1. 계산 결과 출력

`검증기`
1. 구분자 검증
2. 숫자 검증

`구분기`
1. 구분자 종류 판별
2. 구분자 추출
3. 숫자 추출

`계산기`
1. 숫자 합계

### 예외 상황

1. 빈문자열은 0을 반환한다.

2. 음수 입력 시 오류가 발생한다.

3. 커스텀 구분자가 숫자일 경우 오류가 발생한다.

4. 커스텀 구분자와 구분자가 일치하지 않을 경우 오류가 발생한다.

5. 공백은 오류를 발생한다.
